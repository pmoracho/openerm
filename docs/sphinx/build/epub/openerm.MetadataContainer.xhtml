MetadataContainer

Un contenedor de metadatos es un tipo de «bloque» de un Database OpenErm, representa los atributos que describen el reporte. Cada reporte tiene un conjunto básico y fijo de atributos y es posible agregar los que deseemos.

add(extradata)[fuente]

Parámetros

Ejemplo:
>>> import datetime
>>> from openerm.MetadataContainer import MetadataContainer
>>> now = datetime.datetime.now()
>>> m = MetadataContainer("Reporte sin identificar", "n/a", "n/a", "n/a", now)
>>> extra_data = {"autor": "Ernesto Guevara", "Estado": "Draft"}
>>> m.add(extra_data)
dump()[fuente]

Retorna en bytes el contenido de los metadatos ya listos para comprimir y eventualmentre salvar físicamente en el Database Oerm. El «dump» termina siendo una representación plana de un diccionario de datos en formato JSON. Los datos pueden ser variables.

Ejemplo:
>>> import datetime
>>> from openerm.MetadataContainer import MetadataContainer
>>> now = datetime.datetime.now()
>>> m = MetadataContainer("Reporte sin identificar", "n/a", "n/a", "n/a", now)
>>> data = m.dump()
>>> print(data)
b'{"departamento": "n/a", "fecha": "20160914", "aplicacion": "n/a", "sistema": "n/a", "reporte": "Reporte sin identificar"}'

Nota

El método dump entrega una estructura como el siguiente ejemplo:

+=============+
|    Datos    | --> Comprimibles
|  Json dump  |
+=============+
load(data)[fuente]

Recupera de un conjunto de bytes los metadatos

Parámetros

data – bytes

Ejemplo:
>>> import datetime
>>> from openerm.MetadataContainer import MetadataContainer
>>> now = datetime.datetime.now()
>>> m = MetadataContainer("Reporte sin identificar", "n/a", "n/a", "n/a", now)
>>> data = m.dump()
>>> print(data)
b'{"departamento": "n/a", "fecha": "20160914", "aplicacion": "n/a", "sistema": "n/a", "reporte": "Reporte sin identificar"}'
>>> m2 = MetadataContainer()
>>> m2.load(data)
>>> print(m2)
{'departamento': 'n/a', 'fecha': '20160914', 'sistema': 'n/a', 'aplicacion': 'n/a', 'reporte': 'Reporte sin identificar'}
