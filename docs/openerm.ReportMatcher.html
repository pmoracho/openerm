ReportMatcher¶

ReportMatcher es el objeto que identifica los reportes de una determinada cola de impresión. Lo hace mediante una serie de reglas que se definene en una arcchivo de configuración en formato YAML

class openerm.ReportMatcher.ReportMatcher(configfile='openerm.cfg')[fuente]¶

Clases base: object

Matcher de reportes

Parámetros:configfile (string) – Path absoluto al archivo de configuración de la clase

Example

>>> from openerm.ReportMatcher import ReportMatcher
>>> r = ReportMatcher("../var/reports.yaml")
>>> print(r.match("R8101611"))
